# Faceless Shorts Automation System (Execution-Only)

Goal:
Build a faceless, short-form content system using free tools, optimized for consistency, retention, and future monetization.

Platforms:
- YouTube Shorts
- Instagram Reels

Content Type:
- Faceless
- AI-assisted
- Shorts only (9:16)

---

## Weekly Niche Schedule

One niche per day. Same format, same voice, same style.

Monday    : Dark Psychology (Post)
Tuesday   : AI (Post)
Wednesday : Tech (Post)
Thursday  : Batch + Prep
Friday    : Motivational Stories (Post)
Saturday  : Gym / Discipline (Post)
Sunday    : Review + Rest

Rules:
- Never mix niches on the same day
- No skipping posting days

---

## Posting Volume

- Minimum: 2 shorts per posting day
- Recommended: 3 shorts
- Weekly total: 16–20 shorts
- Same videos reposted on Instagram Reels

---

## Video Structure (Mandatory)

1. Hook (0–2s): curiosity, fear, or authority
2. Core (3–20s): single idea only
3. Loop (last 2s): open-ended or tease

---

## Tools (Free Only)

Scripts & Hooks:
- ChatGPT

Voiceover (choose one, stay consistent):
- TTSMaker
- ElevenLabs (free tier)
- PlayHT (free tier)

Editing:
- CapCut (auto-captions, bold text, minimal effects)

Background Clips:
- Pexels
- Pixabay
- Mixkit

---

## Daily Workflow

1. Generate hook + 30–50 word script
2. Convert script to voice (TTS)
3. Import voice into CapCut
4. Add background clip
5. Apply auto-captions
6. Export 1080x1920

Average time per video after setup: ~5 minutes

---

## Batch Day (Thursday)

- Generate scripts for next week
- Record all voiceovers
- Create at least 10 videos
- Save drafts for posting days

---

## Captions & Hashtags

Caption:
- One curiosity-driven sentence
- Follow CTA

Hashtags:
- Maximum 3
- Niche-specific only

---

## Analytics (Sunday Only)

Track:
- Views
- Average watch time
- Retention
- Shares

Do not delete videos or change niches mid-week.

---

## Monetization Path

Phase 1 (0–1k followers):
- No monetization
- Focus on consistency and retention

Phase 2 (1k–10k):
- Affiliate links (books, tools, gear)
- Link-in-bio setup

Phase 3 (10k+):
- Own digital product (PDF, Notion template)
- Tool affiliates
- Brand deals

---

## Non-Negotiable Rules

- Same font, same voice, same layout
- No niche mixing
- No skipped posting days
- Minimum commitment: 90 days
- Low views are not failure
- Inconsistency is failure
